Ввод исходных данных

Исходные данные ПО Lime задаются в табличной форме в предназначенных для этого разделах программы:

  • Узлы
  • Оборудование
  • Ветви
  • Магнитосвязи

Перед тем как приступать к вводу данных, рекомендуется подготовить исходную схему замещения сети – см. раздел “Предварительная подготовка”.

Узлы [in_nodes]

Для узлов схемы замещения предусмотрено задание следующих параметров:

  • Состояние
    Переключатель, задающий состояние узла (включен/отключен).

  • Наименование
    Название узла (произвольный текстовый идентификатор).

    Допускается использование одинаковых названий (программа выдаст обычное предупреждение), однако для наглядности лучше присваивать узлам уникальные текстовые идентификаторы. Наиболее простой вариант – использовать в названиях числовые обозначения (см. рисунок ниже).

  • Uном, [кВ]
    Номинальное напряжение в узле электрической сети (действительное число).

Оборудование [in_equip]

Для оборудования схемы замещения предусмотрено задание следующих параметров:

  • Состояние
    Переключатель, задающий состояние устройства (включено/отключено).

  • Узел
    Наименование существующего узла сети, в котором расположено устройство (выбирается из списка имеющихся узлов).

  • Дисп. наименование
    Название устройства (произвольный текстовый идентификатор).

  • R, [Ом]
    Активное сопротивление прямой последовательности (действительное число).

  • X, [Ом]
    Индуктивное сопротивление прямой последовательности (действительное число).

  • R2, [Ом]
    Активное сопротивление обратной последовательности (действительное число).

  • X2, [Ом]
    Индуктивное сопротивление обратной последовательности (действительное число).

  • R0, [Ом]
    Активное сопротивление нулевой последовательности (действительное число).

  • X0, [Ом]
    Индуктивное сопротивление нулевой последовательности (действительное число).

  • ЭДС, [кВ]
    Действующее значение ЭДС (действительное число).

  • ЭДС угол, [град.]
    Угол поворота вектора ЭДС на комплексной плоскости (действительное число).

Ветви [in_branches]

Для ветвей схемы замещения предусмотрено задание следующих параметров:

  • Состояние
    Переключатель, задающий состояние ветви (включена/отключена).

  • Узел начала
    Наименование узла начала ветви (выбирается из списка имеющихся узлов).

  • Узел конца
    Наименование узла конца ветви (выбирается из списка имеющихся узлов).

  • Дисп. наименование
    Название ветви (произвольный текстовый идентификатор).

    Если оставить данное поле пустым, название будет сформировано автоматически в соответствии шаблоном: “Наименование узла начала - Наименование узла конца”.

  • R1 (R2), [Ом]
    Продольное активное сопротивление прямой (обратной) последовательности (действительное число).

  • X1 (X2), [Ом]
    Продольное индуктивное сопротивление прямой (обратной) последовательности (действительное число).

  • B1 (B2), [мкСм]
    Суммарная реактивная проводимость прямой (обратной) последовательности (действительное число).

  • G1 (G2), [мкСм]
    Суммарная активная проводимость прямой (обратной) последовательности (действительное число).

  • Uном Узел начала, [кВ]
    Напряжение в узле начала ветви, используемое для расчета коэффициента трансформации (действительное число). Для ветвей без коэффициентов трансформации можно не задавать.

  • Uном Узел конца, [кВ]
    Напряжение в узле конца ветви, используемое для расчета коэффициента трансформации (действительное число). Для ветвей без коэффициентов трансформации можно не задавать.

  • Трансформатор угол, [град.]
    Угол комплексного коэффициента трансформации ветви (действительное число).

Параметры схемы замещения нулевой последовательности задаются после установки переключателя в верхней части окна:

  • R0, [Ом]
    Продольное активное сопротивление нулевой последовательности (действительное число).

  • X0, [Ом]
    Продольное индуктивное сопротивление нулевой последовательности (действительное число).

  • B0, [мкСм]
    Суммарная реактивная проводимость нулевой последовательности (действительное число).

  • G0, [мкСм]
    Суммарная активная проводимость нулевой последовательности (действительное число).

  • Сост. ветви нулевой послед. со стор. узла начала
    Переключатель, задающий состояние узла начала ветви в схеме замещения нулевой последовательности (заземлено/не заземлено/отключено).

  • Сост. ветви нулевой послед. со стор. узла конца
    Переключатель, задающий состояние узла конца ветви в схеме замещения нулевой последовательности (заземлено/не заземлено/отключено).

Магнитосвязи [in_mutual_induction]

Для ветвей со взаимной индукцией предусмотрено задание следующих параметров:

  • Состояние
    Переключатель, задающий состояние связи (включена/отключена).

  • Ветвь 1
    Наименование первой ветви (выбирается из списка имеющихся ветвей).

  • Ветвь 2
    Наименование второй ветви (выбирается из списка имеющихся ветвей).

  • R, [Ом]
    Взаимное активное сопротивление ветвей (действительное число).

  • X, [Ом]
    Взаимное индуктивное сопротивление ветвей (действительное число).
