О программе

Программное обеспечение Lime предназначено для расчета токов короткого замыкания в соответствии с действующим документом "Руководящие указания по расчету токов короткого замыкания и выбору электрооборудования" (РД 153-34.0-20.527-98).

Особенности программного обеспечения

  1. Расчет токов короткого замыкания на основе метода симметричных составляющих.

  2. Возможность пакетного расчёта для группы узлов и вида короткого замыкания.

  3. Учет переходного сопротивления в точке короткого замыкания (дуговое замыкание).
