Подготовка исходных данных

Перед тем как приступать к расчетам, необходимо подготовить исходные данные:

  1. Составить расчетную схему сети, соответствующую выбранным расчетным условиям. В общем случае такая схема может включать такие элементы, как:

    • шины;
    • линии электропередачи (воздушные и кабельные);
    • силовые выключатели;
    • трансформаторы;
    • генераторы;
    • нагрузки;
    • шунтирующие реакторы;
    • батареи статических конденсаторов;
    • токоограничивающие реакторы.

    Пример расчетной схемы приведен на рисунке ниже.

    При составлении схемы использованы следующие обозначения: Г – генератор; Т – трансформатор; АТ – автотрансформатор; ВЛ – воздушная линия электропередачи; Н – обобщенная нагрузка; С – система.

  2. Составить схему замещения прямой последовательности, включающую все элементы исходной расчетной схемы. При составлении схемы следует руководствоваться рекомендациями, приведенными в разделе "Схемы замещения элементов сети".

  3. Присвоить всем имеющимся на схеме замещения узлам (включая промежуточные узлы, добавленные в процессе составления схемы замещения) уникальные наименования и указать номинальные напряжения в именованных единицах.

  4. Определить параметры элементов схемы замещения, используя каталожные данные оборудования и расчетные выражения, приведённые в разделе "Схемы замещения элементов сети":

    • Для линий электропередачи – значения комплексных продольных сопротивлений и проводимостей на землю.

    • Для трансформаторов – значения комплексных сопротивлений и комплексных проводимостей Г-образной схемы замещения, приведенные к ступени напряжения, соответствующей узлу начала трансформаторной ветви.

      Определить напряжения узлов начала и конца ветви в соответствии с одним из вариантов: напряжения холостого хода обмоток высшего и низшего напряжения (при необходимости, с учетом фактических регулировочных ответвлений) для точного учета коэффициента трансформации; средние номинальные напряжения для приближенного учета коэффициента трансформации.

      Для трехобмоточных трансформаторов, автотрансформаторов и двухобмоточных трансформаторов с расщепленной обмоткой низшего напряжения, представленных на схеме замещения многолучевой звездой с промежуточным узлом, определить напряжения узлов начала и конца для ветвей с коэффициентами трансформации.

    • Для синхронных машин (генераторы, компенсаторы, электродвигатели) – комплексные сопротивления и ЭДС.

    • Для обобщенных нагрузок – комплексные сопротивления и, при необходимости, ЭДС.

    • Для батарей статических конденсаторов и шунтирующих реакторов – комплексные сопротивления.

    • Для токоограничивающих реакторов – комплексные сопротивления.

  5. Определить ЭДС и сопротивление эквивалента энергосистемы в соответствии с приведенной в документации методикой.

  6. Используя схему замещения и её параметры, ввести исходные данные в предназначенных для этого разделах программы:

    • Данные по узлам схемы – в разделе "Узлы".

    • Данные по синхронным машинам, нагрузкам, шунтам и эквиваленту системы – в разделе "Оборудование".

    • Данные по линиям электропередачи, силовым выключателям, трансформаторам и токоограничивающим реакторам – в разделе "Ветви".

    • Данные по индуктивно связанным ветвям – в разделе "Магнитосвязи".

Выполнение расчетов токов короткого замыкания рассмотрено в разделе "Расчет" документации.
